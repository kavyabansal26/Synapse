# 🚀 Synapse – AI-Powered Studio Application

Synapse is a modern AI-powered studio application built using **React 19**, **Vite**, and the **Google Gemini API**. It enables users to build, interact with, and deploy intelligent features through a sleek and responsive web interface.

Designed for performance and scalability, Synapse integrates real-time data visualization, smooth UI interactions, and a robust backend powered by Firebase.

---

## ✨ Features

* 🤖 **AI-Powered Capabilities**
  Integrated with Google Gemini API for intelligent responses and automation.

* ⚛️ **Modern Frontend**
  Built with React 19 and TypeScript for scalability and type safety.

* 📊 **Real-Time Data Visualization**
  Interactive charts and analytics powered by Recharts.

* 📁 **File Upload Support**
  Drag-and-drop functionality using react-dropzone.

* 🎨 **Responsive UI**
  Styled with Tailwind CSS for a clean and modern design.

* 🎬 **Smooth Animations**
  Enhanced user experience with Motion.js.

* 🔥 **Backend Integration**
  Firebase for authentication, database, and hosting.

## ⚙️ Installation & Setup

1. **Clone the repository**

```bash
git clone https://github.com/your-username/synapse.git
cd synapse
```

2. **Install dependencies**

```bash
npm install
```

3. **Add environment variables**
   Create a `.env` file in the root and add:

```
VITE_GEMINI_API_KEY=your_api_key
VITE_FIREBASE_CONFIG=your_firebase_config
```

4. **Run the development server**

```bash
npm run dev
```
